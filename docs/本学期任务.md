# 一、基本任务和目标：

(1)  完善上学期的深度学习框架，形成系统级别的框架包括：

A.	模型训练过程的可视化模块完善
B.	数据处理模块（含数据增强，随机裁剪、旋转、颜色抖动、MIXup和CutMix算法
C.	模型的保存（JSON+NPZ方式，可以拓展ONNX方式）、
D.	模型的推理。包括模型量化（从浮点数到INT8）
E.	服务化部署模块（gRPC通信），
F.	模型的指标评价模块

(2)  卷积和池化基本实现和准确度验证。卷积能够实现Im2col+gemm的模式。

(3)  改进图像分类任务效果：基于Donkeycar模拟器上的基本场景，继续完善搜集数据集，进行数据处理、进行Vgg的训练并得到分类结果（此过程需要在其他框架Pytorch/ TensorFlow/PaddlePaddle上同时进行，比较同等条件下的不同框架的训练时间、资源占用、FLOPS等等的指标）。

(4)  完成深度学习框架的总体设计、模块设计、算法设计、详细设计等内容，并完成报告撰写。



# 二、可选拓展思路：

(1)	框架算法优化：CNN卷积效率提升（不止Im2col）、或者更加进阶的基于计算图的各种优化、以及其他自选的算法优化（例如Winograd卷积算法，例如数据重排优化循环分块（Loop Tiling）提高 L1/L2 Cache 命中率）。
(2)	模型算法优化：批归一化&残差模块&正则化&权重初始化，基于残差模块实现ResNet（至少是18层）并应用于图像分类任务
(3)	计算图优化：算子融合等，计算图调度也可以是一个思路（更难）
(4)	框架结构优化：数据处理模块与模型训练并行（生产者消费者模型）
(5)	框架结构优化：Docker封装服务和Kubernetes管理服务（Kuflow与框架结合调度任务算子）。
(6)	其他同学自己阅读文献思考的优化内容。
